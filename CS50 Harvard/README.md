# How to earn CS50 certificates

This is a simple tutorial to get free CS50 certificates.

## Steps

### 1. Register on websites

First of all, you must register on the [GitHub](https://github.com) website and then create accounts on other websites with your GitHub account. To create a GitHub account, you must visit there website and sign up.

You also need to create an account on the [Edx](https://www.edx.org) website to earn the certificate.

To see your progress in each course, you must log in to the [cs50.me](https://cs50.me) website with your GitHub account.

Finally, you need to log in to the [cs50.dev](https://cs50.dev) website with your GitHub account to have a codespace like vscode.

### 2. Select the Course

Now you need to choose the course you want. From this [link](https://cs50.harvard.edu/x/courses/) you can see the different courses that CS50 offers.

When you click on any of the courses, you will enter its dedicated page. In the left panel, you can see the exercises. When you click on any of them, at the bottom of the page, when you scroll, you will reach a link called Problem Set or Project. By clicking on these links, you will enter the page to submit the exercises for that course.

### 3. Submit problem sets

Now you have two options:

- You can go and search the name of the course you want on YouTube and see the related tutorials.
- You can send problem sets and projects directly without seeing the tutorials. (For those who know the topic of that course)

To submit problem sets and projects, you must go to cs50.dev and complete each exercise given for that course one by one and submit it. At the bottom of each problem set page, there are methods for checking whether the exercise is correct or incorrect and how to submit it.



### 4. Earn certificate

Finally, after all the exercises have been submitted and the exercises are 100% complete on the cs50.me website, you will be shown a green message on this page that you can get your certificate by clicking on "Click here for your CS50 Certificate".
